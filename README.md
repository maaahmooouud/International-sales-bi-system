# International Sales BI System (End-to-End)

## 📌 Project Overview
An automated, production-ready Business Intelligence (BI) and Data Engineering solution built to solve financial reporting fragmentation in international retail. The system centralizes raw multi-currency transaction streams, handles data quality anomalies, unifies volatile currencies into a standardized baseline asset, and builds an interactive ecosystem for executive financial monitoring.

## ⚙️ Tech Stack & Architecture
* **Data Warehouse:** SQL Server (Relational database structure, staging environments, and core analytical query hosting).
* **ETL Pipeline:** SQL Server Integration Services (SSIS) (Automated pipelines, advanced multi-conditional Lookup transformations for instantaneous exchange-rate mapping, and custom derived columns).
* **Data Visualization:** Power BI Desktop (Dynamic metrics, analytical distribution layouts, and responsive cross-filtering slicers).

## 📊 Key Features
* **Dynamic Currency Standardization:** Automated transformation of multi-currency transactions (DZD, EUR, SAR) into a consolidated USD metric.
* **Enterprise Data Quality:** Integrated data cleansing modules to intercept NULL exceptions, eliminate operational duplicates, and enforce schema compliance.
* **Interactive Financial Dashboarding:** High-impact strategic overview containing Global revenue KPIs, shipping overhead analytics, and localized market composition metrics.
